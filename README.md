# 🚀 Railway Hosting Bot

Telegram-бот для хостинга Railway. Личный бот с системой отзывов, FAQ, панелью администратора.

## ✨ Функции

- ⭐ Система отзывов (1–5 звёзд + текст)
- 📋 Частые вопросы
- 📩 Написать администратору
- 🌐 Контакты / Telegram-канал
- 🔧 Панель администратора:
  - Просмотр всех пользователей
  - Просмотр всех отзывов со средней оценкой
  - Массовая рассылка
  - Добавление/удаление администраторов

## 📁 Структура файлов
```
bot.py          — основной код бота
requirements.txt — зависимости Python
Procfile        — команда запуска
railway.toml    — конфиг Railway
runtime.txt     — версия Python
data.json       — база данных (создаётся автоматически)
```
